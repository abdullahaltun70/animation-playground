# Features

Animation Playground comes packed with powerful features designed to make creating web animations simple, intuitive, and fun.

## üé® Animation Creation

### Intuitive Configuration Panel

- **Visual Controls**: Easy-to-use sliders, dropdowns, and input fields
- **Real-time Updates**: See changes instantly as you adjust parameters
- **Animation Types**: Support for fade, slide, scale, rotate, and bounce animations
- **Timing Controls**: Fine-tune duration, delay, and easing functions

### Live Preview

- **Instant Feedback**: Watch your animations play in real-time
- **Interactive Testing**: Play, pause, and replay animations on demand
- **Responsive Preview**: See how animations look on different screen sizes
- **Side-by-side View**: Configuration panel and preview work together seamlessly

<Callout type="tip">
  The live preview updates immediately as you change settings, so you can
  experiment freely without any delays!
</Callout>

### User Accounts

- **Secure Authentication**: Sign up and log in using Supabase Auth
- **Personal Library**: Save and organize your animation configurations
- **Cloud Storage**: Access your animations from any device
- **Version History**: Keep track of changes to your animations

### Configuration Management

- **Save Configurations**: Store your animation settings for later use
- **Custom Names**: Give descriptive names to your animation configs
- **Quick Access**: Browse through your saved animations easily
- **Duplicate & Modify**: Copy existing configs as starting points

## üîó Sharing & Export

### Shareable Links

- **Unique URLs**: Each animation configuration gets its own shareable link
- **Public Access**: Share animations with anyone, even without an account
- **Read-only Mode**: Viewers can see and test animations but not modify them
- **Copy Protection**: Original creators maintain ownership

### Code Generation

- **React Components**: Export animations as ready-to-use React components
- **Pure CSS**: Generate CSS-only animations for non-React projects
- **Multiple Formats**: Choose the export format that fits your workflow
- **Copy to Clipboard**: One-click copying of generated code
