## What is Animation Playground?

Animation Playground is a Next.js application designed to help you create, preview, and share web animations with ease. Perfect for both novice and experienced developers!

The application leverages the `animation-library-test-abdullah-altun` library to power its animation capabilities, offering a seamless experience for creating beautiful web animations through an intuitive interface.

## Animation Library - Abdullah Altun

A lightweight, performant, and customizable animation library for React applications, powered by SCSS and CSS Custom Properties. Easily add entrance animations to your components.

## Built with Modern Technologies

<Tabs groupId="docs-type">
<TabItem label="üéÆ Playground App" value="playground">

Animation Playground is built using cutting-edge technologies to ensure performance, scalability, and developer experience:

- **[Next.js](https://nextjs.org/)** - React framework with server-side rendering
- **[TypeScript](https://www.typescriptlang.org/)** - Type-safe JavaScript development
- **[Supabase](https://supabase.com/)** - Backend-as-a-Service for authentication and database
- **[Radix UI](https://www.radix-ui.com/)** - Accessible and unstyled UI components
- **[Vitest](https://vitest.dev/)** - Fast and modern testing framework

Ready to get started? Head over to our [Quick Start guide](https://docs.page/abdullahaltun70/animation-playground/setup) to set up your development environment.

</TabItem>
<TabItem label="üì¶ Animation Library" value="library">

The animation library is built with modern web standards and best practices:

- **[React](https://reactjs.org/)** - Modern UI library with hooks support
- **[TypeScript](https://www.typescriptlang.org/)** - Full type safety and IntelliSense
- **[SCSS](https://sass-lang.com/)** - Advanced CSS with variables and mixins
- **[CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/--*)** - Runtime animation control
- **[Hardware Acceleration](https://developer.mozilla.org/en-US/docs/Web/CSS/transform)** - Smooth 60fps animations
