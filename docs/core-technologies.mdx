# Core Technologies

<Info>
  Animation Playground is built with modern, production-ready technologies that
  ensure performance, maintainability, and developer experience.
</Info>

<CardGroup cols={2}>
<Card title="Next.js" icon="react">
**React Framework**

React framework for server-side rendering, static site generation, and optimal performance with built-in routing and API support.

</Card>
<Card title="React" icon="atom">
**UI Library**

JavaScript library for building interactive user interfaces with component-based architecture and state management.

</Card>
</CardGroup>

<CardGroup cols={2}>
<Card title="TypeScript" icon="typescript">
**Type Safety**

Superset of JavaScript that adds static typing for better code reliability, developer experience, and IDE support.

</Card>
<Card title="Animation Library" icon="magic">
**Core Animations**

Custom `animation-library-test-abdullah-altun` package providing the core animation functionalities and React components.

</Card>
</CardGroup>

<CardGroup cols={2}>
<Card title="Supabase" icon="database">
**Backend Services**

Open-source Firebase alternative providing database, authentication, real-time subscriptions, and edge functions.

</Card>
<Card title="Drizzle ORM" icon="server">
**Database Layer**

TypeScript-first ORM for type-safe database interactions with SQL databases and schema management.

</Card>
</CardGroup>

<CardGroup cols={2}>
<Card title="Radix UI" icon="components">
**Component Library**

Unstyled, accessible UI primitives that provide the foundation for custom-designed, accessible user interfaces.

</Card>
<Card title="Vitest" icon="check-circle">
**Testing Framework**

Fast and modern testing framework with TypeScript support, built-in mocking, and excellent developer experience.

</Card>
</CardGroup>

<Card title="Code Quality" icon="code">
**Development Tools**

- **ESLint**: Code linting for identifying and fixing code quality issues
- **Prettier**: Automatic code formatting for consistent code style
- **TypeScript**: Compile-time type checking and IntelliSense support

<Success>
These tools ensure code quality, consistency, and maintainability across the entire project.
</Success>
</Card>
